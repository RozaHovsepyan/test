ԻՄ ՄԱՍԻՆ

Ես Ռոզան եմ: Ծնվել եմ 2007 թվականի մարտի 24-ին, 15 տարեկան եմ: Սովորում եմ Մրգավանի միջնակարգ դպրոցի 9-րդ դասարանում: Սիրում եմ նկարել: Հաճաախում եմ IT Արտաշատ ծրագրավորման կենտրոն: Դպրոցում իմ ամենասիրելի առարկան գրականությունն է: Հետաքրքվում եմ K-POP երաժշտությամբ: Սիրում եմ K-POP խմբեր:Սիրում եմ զբոսնել, զվարճանալ ընկերներիս հետ , մենակ մնալ: Ազատ ժամանակ նկարում եմ և ֆիլմեր դիտում:Ունեմ նաև շատ երազանքներ, նպատակներ … Իմ կարծիքով բոլոր մարդիկ տարբեր են, ես էլ եմ յուրահատուկ մարդ՝ իմ լավ ու վատ կողմերով: Շատ ընկերասեր եմ, սիրում եմ նոր ընկերներ ձեռք բերել, շուտ եմ նվիրվում, բայց և շուտ հիասթափվում եմ, ամեն փոքրիկ բանից մեծ ուրախություն եմ ապրում, ցավոք շատ նեղացկոտ եմ, զգացմունքային, շատ եմ սիրում լինել ուրախ, միշտ ժպտալ, լինում եմ նաև չափազանց լուրջ իմ կարծիքով խորը մարդ եմ … Գլխավոր նպատակս՝ կյանքն ապրեմ վառ ու լիարժեք …

ԻՆՉ ԾՐԱԳՐԱՎՈՐՄԱՆ ԼԵԶՈՒՆԵՐԻ ԵՄ ՏԻՐԱՊԵՏՈՒՄ

Տրապետում եմ 3 ծրագրավորման լեզվի՝ HTML , CSS , JS (JavaScript) : 2022 թվականի մարտ ամսից հաճաախում եմ IT Արտաշատ ծրագրավորման կենտրոն ։ Սկզբում սովորել եմ Scratch խաղերի ծրագրավորում,իսկ հետո սկսել եմ սովորել Web ծրագրավորում ։ Սկզբում սովորել եմ HTML ծրագրավորման լեզուն , այնուհետև CSS և JS (JavaScript)։
